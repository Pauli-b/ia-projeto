# Detecção de Mão com MediaPipe + OpenCV

Projeto em Python para detecção de mão e contagem de dedos levantados usando MediaPipe e OpenCV.
O sistema identifica apenas uma mão por vez e atualiza dinamicamente uma imagem na tela, indicando quantos dedos estão levantados (0 a 5).

## Funcionalidades

- Detecção de mão em tempo real usando MediaPipe Hands.

- Contagem de dedos levantados (0 a 5).

- Exibição automática de imagens correspondentes ao número de dedos mostrados.

- Retângulo verde exibindo o número detectado.

- Suporte a captura por webcam.

## Tecnologias Utilizadas

Python 3.10+
MediaPipe
OpenCV
